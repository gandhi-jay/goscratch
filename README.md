# Go Scratch

A repository for learning Go by building real projects and practicing Go concepts in practice.

## Getting Started

### Prerequisites

- Go 1.21 or higher installed
- Git

### Installation

Clone the repository:

```bash
git clone https://github.com/gandhi-jay/goscratch.git
cd goscratch
```

### Running Examples

Each directory contains standalone examples. Run them with:

```bash
cd <directory-name>
go run main.go
```

## Topics Covered

- Basic syntax and data types
- Functions and methods
- Interfaces
- Concurrency (Goroutines, Channels)
- Error handling
- Testing
- Standard library usage
